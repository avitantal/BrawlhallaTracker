BRAWLHALLA Session Tracker
‏טראקר חד־קובץ במלואו (HTML יחיד, ללא בנייה, ללא שרת) למשחקי Brawlhalla עם חברים. נפתח בדפדפן ושומר הכל ב־localStorage.

‏פיצ'רים מרכזיים
‏בריכת לוחמים עם אווטארי אמוג'י מותאמים אישית.
‏סשנים מקבילים — אפשר לפתוח כמה סשנים בו־זמנית ולעבור ביניהם בלחיצה על צ'יפ.
‏לוח תהילה חי עם תוארי "אלוף האלופים" / "זבל הזבלים".
‏מדד ההומואים — לוח ייעודי עם הומו ההומואים הנוכחי וגרף נרות צבעוני שמציג את כמות הפעמים שכל לוחם היה הומו.
‏היסטוריית משחקים עם צ'יפ מוד צבעוני, סדר סיום, וטאגי נוקאאוט פר שחקן.
‏ארכיון סשנים — סשנים שהסתיימו נשמרים עם הדירוג וההיסטוריה המלאה.
